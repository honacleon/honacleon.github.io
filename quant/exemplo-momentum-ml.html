Momentum ML — Sinais de Direção Intradiário

Classificação da direção do próximo candle com gradient boosting, backtest e gestão de risco por volatilidade.

Notebook somente leitura. O código está visível para fins de portfólio, mas a base de dados é privada: a conexão usa variável de ambiente (nunca credenciais no código), e os números exibidos vêm de uma amostra sintética. Nada aqui executa contra dados reais.

# Visão geral

A estratégia trata a previsão de direção como um problema de classificação binária: a cada candle, o modelo estima se o próximo fechamento será maior (1) ou menor (0) que o atual. As features combinam momentum, volatilidade e microestrutura. O dimensionamento de posição é proporcional ao inverso da volatilidade.

# Feature engineering

Retorno, momentum de 12 períodos, volatilidade móvel (20) e RSI(14). O alvo é a direção do próximo candle.

def make_features(df):
    df["ret"]    = df["close"].pct_change()
    df["mom"]    = df["close"].pct_change(12)
    df["vol"]    = df["ret"].rolling(20).std()
    df["rsi"]    = rsi(df["close"], 14)
    df["target"] = (df["close"].shift(-1) > df["close"]).astype(int)
    return df.dropna()

data = make_features(prices)

# Treino do modelo

Split temporal 70/30 (sem embaralhar, para não vazar futuro). Gradient boosting com regularização leve.

features = ["ret", "mom", "vol", "rsi"]
split = int(len(data) * 0.7)

X_te, y_te = data[features][split:], data["target"][split:]

model = xgb.XGBClassifier(n_estimators=400, max_depth=4, learning_rate=0.03)

# Backtest e gestão de risco

Posição long quando o sinal é 1, dimensionada pelo inverso da volatilidade. Métricas calculadas sobre o período de teste (out-of-sample).

In [4]:
signal = model.predict(X_te)
ret    = data["ret"][split:].values
size   = 1 / data["vol"][split:].values     # risco-alvo por volatilidade
pnl    = signal * ret * np.clip(size, 0, 3)
equity = (1 + pnl).cumprod()
dd     = equity / np.maximum.accumulate(equity) - 1

print(f"PnL total : {(equity[-1]-1)*100:+.1f}%")
print(f"Max DD    : {dd.min()*100:.1f}%")
